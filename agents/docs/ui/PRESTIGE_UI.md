# UI do Prestígio

## Elementos necessários
- card de status do conhecimento
- painel de prestígio com requisito atual
- botão de prestigiar
- confirmação explicando perdas e ganho
- feedback visual quando o prestígio ficar disponível

## Objetivo de clareza
O jogador deve entender rapidamente:
- que existe progressão permanente
- quando ela está desbloqueada
- que o reset é opcional
- que o bônus é permanente e ligado às vendas

## Feedback proposto
- HUD:
  - nível atual
  - bônus permanente atual
- painel:
  - requisito atual
  - resumo do que acontece ao prestigiar
- toast:
  - avisar quando o prestígio ficar disponível na run atual
