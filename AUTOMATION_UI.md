# Automation UI

## Feedback necessário
- botão de compra com custo claro
- estado ativo do helper após a compra
- indicador no HUD mostrando que a automação está ligada
- mensagem curta quando o helper colher
- distinção visual entre colheita manual e automática

## Ajustes de interface
- novo card de melhoria para `Farm Helper`
- novo card no HUD para mostrar status:
  - desligado
  - ativo
  - próximo ciclo
- texto de suporte explicando que o helper colhe plantas prontas automaticamente
- destaque visual breve no canteiro colhido pelo helper
- mensagem de status sem excesso de ruído

## Critérios de clareza
- o jogador deve entender em poucos segundos:
  - como comprar
  - o que o helper faz
  - que ele não vende automaticamente
  - que o combo continua sendo manual
